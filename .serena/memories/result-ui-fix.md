## Work Done
1.  **UI 구조 전면 개편 (Sidebar + Main Content)**:
    *   **좌측 사이드바**: 모든 설정 그룹(경로, 기체, 상세 파라미터, 안전 점검, 옵션, 템플릿)과 액션 버튼(실행, 프리셋)을 좌측 사이드바에 수직으로 배치했습니다. 스크롤바를 추가하여 저해상도 환경에서도 모든 설정에 접근 가능합니다.
    *   **우측 메인 영역**: 상단에는 대형 지도 미리보기를, 하단에는 로그 창을 배치하는 상하 분할 구조(`PanedWindow`)를 도입하여 지도 가독성을 극대화했습니다.
2.  **지도 미리보기 기능 개선**:
    *   **폴리곤 가시성**: `set_path`(선) 대신 `set_polygon`(면/테두리)을 사용하여 미션 영역을 더 명확하게 시각화했습니다. 지도 가독성을 위해 테두리만 강조하고 내부는 반투명(또는 투명)하게 처리했습니다.
    *   **반응형 업데이트**: 기체, 고도, 속도뿐만 아니라 '물리적 버퍼' 값이 바뀔 때도 지도상의 폴리곤 크기가 즉시 갱신되도록 연동했습니다.
3.  **사용 편의성 향상**:
    *   창 크기를 1200x850으로 확장하여 전문적인 GIS 툴에 가까운 넓은 작업 환경을 제공합니다.
    *   로그 영역과 지도 영역의 경계를 마우스로 조절할 수 있습니다.
